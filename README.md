# منظومة التوظيف الذكية متعددة الأبعاد

> تحويل التوظيف من عملية إدارية إلى رحلة حياة مهنية ذكية، مستدامة، ومتطورة — حيث يكون الفرد محور المنظومة، والذكاء الاصطناعي هو المرشد الحيوي.

الواجهة الأمامية (Web) لمنصة توظيف متعددة الأبعاد تجمع الباحثين والجهات والموصين والمدراء في نظام بيئي واحد.

---

## 🛠️ التقنيات

| الطبقة | التقنية |
|--------|---------|
| إطار العمل | **Vue 3** (Composition API, `<script setup>`) |
| اللغة | **TypeScript** (الوضع الصارم) |
| أداة البناء | **Vite 5** |
| واجهة المستخدم | **Vuetify 3** |
| إدارة الحالة | **Pinia** |
| التوجيه | **Vue Router 4** |
| طلبات HTTP | **Axios** |
| التعدد اللغوي | **vue-i18n** (عربي افتراضي + إنجليزي، RTL) |

نظام الألوان مستمدّ من وثيقة الـ UI/UX: أزرق داكن `#1A365D`، فيروزي `#319795`، برتقالي `#ED8936`.

---

## 🚀 التشغيل

```bash
npm install
npm run dev        # خادم التطوير على http://localhost:5173
npm run build      # بناء الإنتاج
npm run typecheck  # فحص الأنواع
```

> **وضع التجربة:** لا يوجد Backend بعد — المصادقة والبيانات محاكاة (Mock). في صفحة الدخول أدخل أي بريد/كلمة مرور.
> لتجربة الأدوار المختلفة اكتب داخل البريد: `company` (جهة توظيف)، `admin` (مدير)، `endorser` (موصٍ)، وإلا فالدور باحث.

## 🗂️ الشاشات المنجزة (المرحلة 0)

| الدور | الشاشات |
|-------|---------|
| **عام (زائر)** | الصفحة التعريفية (Landing)، تسجيل الدخول، إنشاء حساب (اختيار دور) |
| **الباحث** | لوحة التحكم، الملف الشخصي (6 تبويبات)، الفرص (قائمة + تفاصيل + تحليل تطابق)، الرغبات الواردة، مركز التقييم، منشئ السيرة (معالج 7 خطوات)، المساعد الذكي |
| **جهة التوظيف** | لوحة التحكم، نشر فرصة، الترشيحات + ملف المرشح، إدارة الرغبات، التحليلات |
| **الموصي** | لوحة الموصي، إضافة توصية (نص/صوت/فيديو/مستند + أسئلة AI) |
| **المدير** | لوحة المنصة، إدارة المستخدمين، الأدوار والصلاحيات (RBAC) |
| **مشترك** | الاستبيانات الثمانية |

## 📁 البنية

```
src/
├── main.ts, App.vue           # نقطة الدخول + مبدّل الـ Layouts
├── layouts/                   # Default (شريط جانبي حسب الدور)، Forms، Blank
├── modules/<name>/            # وحدات المزايا (pages, components, services, interfaces)
├── components/shared/         # مكوّنات مشتركة (StatCard, PageHeader)
├── plugins/                   # vuetify (ثيم)، i18n، axios
├── router/                    # المسارات + حارس المصادقة
├── stores/                    # AuthStore (Pinia)
└── styles/                    # الأنماط العامة
```
